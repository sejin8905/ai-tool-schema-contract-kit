# Quickstart

Install dependencies:

```bash
npm install
```

Run the full local checks:

```bash
npm test
npm run typecheck
npm run build
```

Run demos:

```bash
npm run dev
npm run demo:fail
npm run demo:diff
```

Run the CLI:

```bash
npm run cli:pass
npm run cli:diff
```

Use the sample JSON files in `examples/` as a starting point for your own tool registry.
